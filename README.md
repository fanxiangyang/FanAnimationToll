# FanAnimationToll
Animation class method

###  功能介绍
##### 功能介绍：封装各种动画的类方法，使用方便，具体使用如下
*   [self.animationImageView.layer addAnimation:[FanAnimationToll fan_scaleMax:2 orginMin:0.1 durTimes:1 Rep:5] forKey:@"scale"];
*   各种动画使用方便，你会喜欢的！
